GUÍA DE DESPLIEGUE — SmartVest (actualizada)
============================================

IMPORTANTE: La validación de direcciones con IA ya NO usa API_KEY en el
navegador. Configura SMARTVEST_GEMINI_API_KEY en api/config.local.php
en tu servidor PHP.

OPCIÓN A — XAMPP (recomendada para proyecto completo: web + API + BD)
--------------------------------------------------------------------
1. Instala XAMPP (Apache + MariaDB).
2. Ejecuta database.sql en phpMyAdmin.
3. En el proyecto: npm install && npm run build && ./scripts/deploy-xampp.sh
4. Abre http://localhost/Smartvest/
5. Para el chaleco ESP32: SMARTVEST_API_URL = http://TU-IP-LAN/Smartvest/api/iot.php

Documentación detallada: docs/INSTALACION.md
Exposición del proyecto: docs/GUIA-EXPOSICION.md
SMS de emergencia (0963930791 cuidador): docs/SMS-SOS-EMERGENCIA.md

OPCIÓN B — Vercel (solo frontend estático)
------------------------------------------
1. Sube el código a GitHub.
2. Importa en https://vercel.com (proyecto Vite).
3. La API PHP debe estar en otro hosting (XAMPP con túnel, VPS, etc.).
4. No configures API_KEY en Vercel para Gemini (obsoleto en cliente).

Documentación: docs/DESPLIEGUE.md

OPCIÓN C — GitHub Pages
-----------------------
Solo sirve archivos estáticos (HTML/JS/CSS). Sin PHP ni MariaDB.
Ver .github/workflows si existe workflow de deploy.
